# TimedAction

[![CI](https://github.com/log0u7/TimedAction/actions/workflows/ci.yml/badge.svg)](https://github.com/log0u7/TimedAction/actions/workflows/ci.yml)

A fork of Alexander Brevig's TimedAction library for Arduino.

Provide an easy way of triggering functions at a set interval.

Maintained by [@log0u7](https://github.com/log0u7).
